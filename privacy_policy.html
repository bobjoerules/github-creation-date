Privacy Policy

Effective Date: March 9, 2026

This Privacy Policy describes how the GitHub Repo Creation Date browser extension ("the Extension") handles your information.

1. Information Collection and Use

The Extension does NOT collect, store, or transmit any personal data, browsing history, or repository information. All processing is performed locally on your device.

2. Data Processing

The Extension reads publicly available metadata from GitHub repository pages solely to display the repository's creation date in the sidebar. This information is never sent to any external servers and remains entirely within your browser's local environment.

3. Permissions

The Extension requires specific permissions to function:

  • Host Permissions (github.com): Necessary to read the repository metadata and inject the date display into the GitHub user interface.
  • activeTab: Used to interact with the repository page you are currently viewing.

4. Third-Party Disclosure

Since the Extension does not collect any data, no information is ever shared with, sold to, or used by third parties.

5. Changes to This Policy

Any future changes to this policy will be updated on this page. However, the core principle of "no data collection" will remain the foundation of the Extension.

6. Contact

If you have any questions about this Privacy Policy, please contact the developer through the extension store's support section.
